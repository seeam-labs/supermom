# 👶 শিশু পরিচর্যা - Baby Care App

একটি সম্পূর্ণ বাংলা শিশু পরিচর্যা ওয়েব অ্যাপ - ০ থেকে ৫ বছর বয়সী শিশুদের জন্য বাংলাদেশি মায়েদের গাইড।

## 🎯 ফিচারসমূহ

### মূল সেকশন
- **🏠 ড্যাশবোর্ড** - শিশুর প্রোফাইল, গ্রোথ স্ট্যাটাস, দ্রুত এক্সেস
- **🍼 খাবার গাইড** - বয়সভিত্তিক খাবারের তালিকা, কখন কি খাওয়াবে
- **📏 গ্রোথ ট্র্যাকার** - ওজন ও উচ্চতা ট্র্যাকিং (আসছে)
- **🏃 এক্টিভিটি** - বয়সভিত্তিক খেলাধুলা (আসছে)
- **😴 ঘুমের গাইড** - ঘুমের সময়সূচি (আসছে)
- **😢 কান্না সমাধান** - শিশুর কান্নার কারণ ও সমাধান (আসছে)
- **🛁 গোসল ও স্বাস্থ্যবিধি** - পরিচ্ছন্নতা গাইড (আসছে)
- **💉 ভ্যাকসিন ট্র্যাকার** - টিকার সময়সূচি (আসছে)
- **🧠 মাইলস্টোন ট্র্যাকার** - বিকাশের মাইলস্টোন (আসছে)
- **✅ চেকলিস্ট হাব** - দৈনিক ও সাপ্তাহিক চেকলিস্ট (আসছে)
- **🚫 সতর্কতা** - জরুরি তথ্য (আসছে)
- **📓 মায়ের ডায়েরি** - নোট ও লগ (আসছে)

### বিশেষ ফিচার
- ✅ একাধিক শিশুর প্রোফাইল
- ✅ localStorage ভিত্তিক ডেটা সেভ
- ✅ ডার্ক/লাইট মোড
- ✅ মোবাইল-ফার্স্ট রেসপন্সিভ ডিজাইন
- ✅ সম্পূর্ণ বাংলা UI
- ✅ Glassmorphism ডিজাইন
- ✅ Vercel-ready স্ট্যাটিক বিল্ড

## 🛠️ টেকনোলজি

- React (Functional Components + Hooks)
- localStorage (ডেটা পার্সিস্টেন্স)
- Recharts (চার্ট)
- Lucide React (আইকন)
- Tailwind CSS (স্টাইলিং)
- Vite (বিল্ড টুল)

## 🚀 ইনস্টলেশন

```bash
npm install
npm run dev
```

## 📦 বিল্ড

```bash
npm run build
```

## 🌐 ডিপ্লয়মেন্ট (Vercel)

1. Vercel এ গিয়ে নতুন প্রজেক্ট তৈরি করুন
2. GitHub রিপোজিটরি কানেক্ট করুন
3. অটোমেটিক ডিপ্লয় হবে

অথবা:

```bash
vercel
```

## 📱 ব্যবহার

1. অ্যাপ ওপেন করলে প্রোফাইল সেটআপ স্ক্রিন আসবে
2. শিশুর নাম, জন্ম তারিখ, লিঙ্গ দিয়ে প্রোফাইল তৈরি করুন
3. ড্যাশবোর্ড থেকে সব তথ্য দেখুন
4. বিভিন্ন সেকশনে গিয়ে বিস্তারিত গাইডলাইন দেখুন

## 💾 ডেটা স্টোরেজ

সব ডেটা localStorage এ সেভ থাকে:
- শিশুর প্রোফাইল
- গ্রোথ ডেটা
- চেকলিস্ট প্রগ্রেস
- ডায়েরি এন্ট্রি
- সেটিংস

ব্রাউজার ক্লিয়ার করলে ডেটা মুছে যাবে।
